```
๋ ࣭ ⭑๋ ࣭ ⭑ - Pengenalan Tag Html (PWB - Gavrila)
1. <!DOCTYPE html> : Memberitahu browser bahwa ini adalah dokumen HTML.
2. <html> : Dengan atribut lang, merupakan akar atau pembungkus utama seluruh halaman.
3. <head> : Tempat menyimpan informasi ""otak"" web (metadata).
4. <body> : Dengan atribut style, merupakan tempat menyimpan semua konten visual yang kita lihat.
5. <h1>   : Judul utama (paling besar).
6. <h2>   : Sub-judul tingkat kedua, (begitupun <h3>-<h6> subjudul yang semakin mengecil).
7. <p>    : Membuat paragraf teks.
8. <b> / <strong> : Menebalkan teks (Strong artinya teks tersebut penting).
9. <i> / <em> : Memiringkan teks (Emphasized artinya teks diberikan penekanan)
10. <u>   : Memberi garis bawah pada teks.
11. <br>  : Pindah ke baris baru (seperti tombol Enter).
12. <hr>  : Membuat garis horizontal pembatas antar bagian.
13. <ul>  : Daftar poin/bullet (Unordered List).
14. <ol>  : Daftar angka 1, 2, 3 (Ordered List).
15. <li>  : Setiap butir dalam daftar.
16. <div> : Kontainer umum untuk mengelompokkan elemen.
17. <span> : Dengan atribut style, mirip <div> tapi untuk bagian kecil di dalam teks (inline).
18. <a>   : Membuat link/tautan menggunakan atribut href.
19. <img> : Menampilkan gambar menggunakan atribut src.
20. <video> : Menampilkan file video di halaman web, menggunakan atribut widht, controls.
21. <audio> : Memutar file suara atau musik, menggunakan atribut src, controls.
22. <form>  : Wadah untuk mengumpulkan input dari pengguna.
23. <label> : Memberikan judul mengenai keterangan kepada pengguna, akan apa yang harus diisi
24. <input> : Kotak tempat pengguna mengetik (teks, email, dll), menggunakan atribut type, placeholder.
25. <button> : Membuat tombol yang bisa diklik.
26. <textarea> : Kotak input teks yang lebih besar (untuk pesan/komentar).
27. <iframe> : dengan atribut src, berfungsi seperti "jendela", yang dapat menampilkan halaman di web lain langsung di halamanmu (mirip seperti <a>)
```
